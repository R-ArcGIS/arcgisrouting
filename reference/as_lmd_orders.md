# Convert spatial objects to Last Mile Delivery orders input

Convert spatial objects to Last Mile Delivery orders input

## Usage

``` r
as_lmd_orders(x, ...)
```

## Arguments

- x:

  An `sf` or `sfc` object containing point geometries representing
  delivery and pickup locations.

- ...:

  Additional arguments passed to methods.

- verbose:

  Logical. Whether to message which recognized attributes are used.
  Default: `TRUE`.
